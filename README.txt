Vanilla Premium Site

Как запустить сайт с настоящим backend:

1. Откройте терминал в этой папке.
2. Запустите:
   node server.js
3. Откройте в браузере:
   http://127.0.0.1:8092

Что теперь реально работает через backend:

- GET /api/stats - текущие активные заказы и забронированные столики.
- POST /api/orders - создание нового заказа из корзины.
- POST /api/reservations - создание бронирования столика.
- GET /api/orders - список заказов.
- GET /api/reservations - список броней.

Данные хранятся в файле data/store.json. Если файла нет, сервер создаст стартовую базу автоматически.
